Требования ГОСТ

  • Шрифт: Times New Roman, 14 пт
  • Междустрочный интервал: 1.5
  • Абзацный отступ: 1.25 см
  • Выравнивание: по ширине
  • Поля: левое - 3 см, правое - 1 см
  • Верхнее/нижнее поле: 2 см
  • Нумерация страниц: снизу по центру
Форматирование по ГОСТ

Текстовый редактор — компьютерная программа, предназначенная для создания и изменения текстовых файлов, а также их просмотра на экране, вывода на печать, поиска фрагментов текста. Основные функции: создание, редактирование, сохранение и печать текста.

Текстовый процессор — приложение для создания и редактирования текстовых документов с расширенными возможностями: форматирование, вставка изображений, таблиц, проверка орфографии. Позволяет создавать профессионально оформленные документы.
